Jmetadata

Complete metadata automatically with MusicBrainz, LastFM or manually.
JMetadata also kwnon as JAudioScobbler is based on last.fm API bindings for Java & MusicBrainz our goal is to create a stand alone application that keeps a good Metadata quality. Metadata is the information about your music files such as genre, cover art, artist, title, track number, cd numb er, etc.
